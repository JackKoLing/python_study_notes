## 项目描述：
    我们要编写一个名为“学习笔记”的Web应用程序，让用户能够记录感兴趣的主题，并在学习每个
主题的过程中添加日志条目。“学习笔记”的主页对这个网站进行描述，用户可以注册、登录或注销。用户
登录后，就可查看自己的数据库。包括：创建新主题、添加或编辑新条目以及阅读既有的条目。
